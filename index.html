Tentang Kami

Selamat datang di website kami, mari bersama-sama belajar memahami dasar bahasa coding sederhana.

History

Berawal dari ketertarikan mengenai bahasa pemrograman, kami memulai belajar ilmu dasar HTML untuk mengenal rangka dasar membuat sebuah website. Terkesan terlambat, akan teteapi belajar hal baru tidaklah terbatas waktu, usia maupun riwayat pendidikan yang di tempuh di bangku sekolah, mengingat perkembangan ilmu teknologi terutama era digitalisasi, belajar tentang bahasa program komputer tidak rugi untuk dilakukan. Tujuan kami mengajak belajar bersama adalah, supaya kita tidak sekedar menjadi pengguna ataupun penikmat teknologi, mari jadikan diri kita lebih kreatif, berkembang dan produktif.

Di mulai dari belajar fungsi kode HTML, atribut yang terpakai, dan banyak hal yang akan menjadi dasar dari struktur sebuah website. Yang selanjutnya ada beberapa tahapan perlu untuk di pelajari setelah paham mengenai HTML, yaitu CSS dan JavaScript.

Latar Belakang

Dengan adanya teknologi AI yang mempermudah pengguna untuk berkreasi di dunia programming, pada akhirnya ilmu dasar pemrograman tidak lagi perlu di kuasai. Dan dari situ maka kami berkemauan mengajak bagi para pemula yang ingin belajar ilmu dasar, agar supaya pondasi dari membangun sebuah kerangka awal tetap di pahami bersama.

Bagi yang mempunyai hobi yang sama, mari sharing dan diskusi untuk memperluas ilmu pengetahuan kita.

Layanan Kami

Berikut adalah deskripsi layanan sharing dan diskusi dari kami.

HTML

Bahasa markah yang digunakan peramban untuk menafsirkan dan menulis teks, gambar dan bahan lainnya kedalam halaman web secara visual maupun suara.

CSS

Bahasa lembar gaya yang digunakan sebagai penentu presentasi dan gaya dokumen yang ditulis dalam bahasa markup, menentukan tampilan dan format web.

JAVASCRIPT

Bahasa pemrograman yang dinamis dan bahasa pemrograman dengan interpretasi untuk berorientasi ke objek dan programing fungsional web interaktif.

PYTHON

Salah satu bahasa pemrograman yang dapat melakukan eksekusi sejumlah instruksi multi guna secara langsung dengan metode orientasi objek.

PHP

Personal Home Page, bahasa skrip sisi server yang di kodekan dengan HTML, untuk mengembangkan situs web dinamis, statis atau aplikasi web.

MySQL

Sistem manajemen database yang bersifat open-source dan menggunakan perintah dasar berupa "structured query language" ang populer didunia teknologi.

HOSTING

Hosting web adalah layanan yang menyimpan situs web atau aplikasi web dan mudah di akses di berbagai perangkat seperti desktop,seluler dan tablet.

WEB SERVER

Salah satu jenis server yang menyediakan layanan permintaan data web. Bekerja dengan cara menyimpan,memproses,dan mengirim file dari website.
